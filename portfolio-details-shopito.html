Shopito – E‑Commerce Platform

Shopito is a full-stack e‑commerce web application built with Java 17, Spring Boot, React.js, PostgreSQL, and Docker. It supports user authentication with JWT, product browsing, cart management, and order placement. CI/CD pipelines ensure smooth deployment, and the system is containerized for scalability. The project demonstrates integration of backend APIs with a modern React frontend, along with DevOps practices for deployment.
